# Chess Sound Effects

Drop your MP3 files in this folder. The game will use them automatically.

Required filenames:
  move.mp3      — piece placed on square (wooden click)
  capture.mp3   — piece captured (louder thud)
  click.mp3     — piece picked up
  correct.mp3   — puzzle move correct
  wrong.mp3     — puzzle move wrong
  win.mp3       — game won
  lose.mp3      — game lost

If a file is missing the game falls back to high-quality
synthesized sounds automatically — no silence, no errors.

Good free sources:
  https://freesound.org (search "chess piece wood")
  https://opengameart.org
  https://pixabay.com/sound-effects
